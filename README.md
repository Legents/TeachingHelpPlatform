# TeachingHelpPlatform
## 简介
智能化课程教学辅助平台主要服务对象为教师和学生，实现教师与学生交互信息的集合。在用户登录后实现信息的智能推送，比如对教师推送学生的作业提交情况、对学生推送作业的批改结果以及新的任务。在学生提交作业后系统可以根据预先设置好的规则对学生提交的作业进行初步的批改并给出结果同时减少教师的工作量。
